Objective

Topics Covered -

- Type Inference
- Primitive Types
- Special Types
- Arrays
- Objects
- Union Types
- Functions

⸻

Requirements

1. Create a students array

Create an array that stores multiple student objects.

Each student should contain:

- id
- name
- marks
- grade

Example student:

{
id: 101,
name: "Omm",
marks: 88,
grade: "A"
}

The id should support both numbers and strings using a union type.

⸻

2. Add Student Function

Create a function that adds a new student to the array.

Example:

Input:
101, "Omm", 88, "A"
Output:
Student added successfully

⸻

3. Print All Students Function

Create a function that prints all students in a readable format.

Example Output:

ID: 101 | Name: Omm | Marks: 88 | Grade: A
ID: 102 | Name: Rahul | Marks: 75 | Grade: B
ID: 103 | Name: Amit | Marks: 92 | Grade: A

⸻

4. Find Student by ID Function

Create a function that accepts a student ID and returns the student details.

Example:

Input:
101
Output:
ID: 101 | Name: Omm | Marks: 88 | Grade: A

If the student does not exist:

Student not found

⸻

5. Calculate Average Marks Function

Create a function that calculates the average marks of all students.

Example:

Marks:
88, 75, 92
Output:
Average Marks = 85

⸻
